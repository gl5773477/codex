# Territory Sandbox

一个上帝视角的大陆演化原型。

## 当前版本

- 六边形大陆地图
- 多个国家初始分布
- 每块领地自动增长人口
- 人口积累后自动向邻地扩张
- 领地被占领后改变颜色
- 上帝视角：暂停、倍速、点击查看、神赐人口

## 本地运行

```bash
cd territory-sandbox
npm install
npm run dev
```

## 技术栈

- React
- TypeScript
- Vite
- Canvas 2D
